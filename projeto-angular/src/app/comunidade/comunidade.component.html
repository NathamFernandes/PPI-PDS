Sobre o Projeto

Olá, e bem-vindo! Essa é a página dedicada ao projeto final da discplina de Programação Para Internet e Projeto de Desenvolvimento de Software. Nela, também abrigará um fórum no futuro, quando os recursos para tal forem implementados.

Mas... Como surgiu o CSBrasil?

O CSBrasil surgiu como projeto final das disciplinas de Programação Para Internet (PPI), ministrada por Bruno Gurgel, e Projeto de Desenvolvimento de Software (PDS), ministrada por Álvaro Hermano, ambas do componente curricular do curso integrado em Informática, no Instituto Federal do Rio Grande do Norte. Fora proposto ao discente a criação de uma aplicação que englobasse grande parte dos conteúdo estudados em sala de aula: o desenvolvimento web de aplicações através do framework Angular, na disciplina de PPI; e a documentação da aplicação, com foco nos requisitos e diagramas do mesmo, na disciplina de de PDS.

Visando um âmbito de maior conforto para com o discente, o tema escolhido para a aplicação se baseou na comunidade de Counter-Strike: Global Offensive (CS:GO) brasileira, uma das maiores comunidades de jogos do Brasil. Para demonstrar tamanha abrangência, uma pesquisa realizada pelo site Leetiffy relatou que, no último mês de julho (07/2022), havia por volta de 14 milhões de jogadores ativos no CS:GO, sendo 5,43% destes apenas da comunidade brasileira do jogo.

A problemática encontrada envolvendo o tema envolve a própria comunidade, a qual, muitas vezes, carece de recursos para maior integração de seus jogadores comuns ao cenário competitivo do jogo. Por esse motivo, os requisitos funcionais do projeto visavam promover a devida experiência aos usuários e solucionar a problemática encontrada, que, de modo mais detalhado, se desenvolveu na falta de uma plataforma informacional sobre o cenário competitivo de CS:GO brasileiro, bem como a indisponibilidade de uma aplicação que mostrasse as estatísticas pessoais dos jogadores do game.

De modo sucinto, a aplicação promove um ambiente que expõe estatísticas e notícias sobre o cenário competitivo do jogo - com foco no âmbito brasileiro -, como também fornece uma página dedicada para consulta e exibição de estatísticas de jogadores comuns - jogadores casuais e não necessariamente competitivos -, tornando escolha do usuário apenas exibi-las ou, de fato, compará-las com outro jogador. O próximo tópico, de descrição da aplicação, abordará um pouco mais sobre a aplicação e seus objetivos, bem como um breve retrospecto sobre as fundamentações técnicas utilizadas no projeto.

Especificações técnicas

No momento o site está hospedado no domínio da Netlify e está recebendo as devidas atualizações ao decorrer do projeto. Desa forma, o projeto está sendo realizado através do framework Angular, em sua versão 14. Para esse projeto, também foram utilizadas APIs para demonstração de informações. A primeira delas, implementada através da biblioteca Node.js hltv-api, possui inúmeras versatilidades e é baseada no site de dados estatísticos de CS:GO, a HLTV.

A segunda delas, é a API da steam criada pelo discente, a qual se conecta à API fornecida pela steam e extrai apenas informações do CS:GO. O motivo pela utilização de uma API que extrai as informações de outra API está relacionado ao bloqueio do CORS, não implementado na API original. Por essa razão, a biblioteca node-fetch cumpre a função da extração de dados anteriormente citada. Além disso, a biblioteca steamapi também converte nomes da steam em IDs, bem como mostra dados do usuário da Steam, de modo a complementar a API criada pelo discente.

Ambas as APIs estão hospedadas na plataforma Cyclic, uma plataforma em nuvem para hospedagem de serviços.
